
# AK-84 Tools

## Краткое описание проекта

Набор полезных вспомогательных скриптов на Python для быстрого предварительного анализа различных наборов данных. Это полностью бесплатный (некоммерческий) проект, который создаётся в свободное время.

## Цели создания проекта

Самые часто нужные функции, которые помогут создавать небольшие домашние проекты с применением датчиков и микроконтроллеров.

## Целевая аудитория

Программисты, тестировщики, системные аналитики, а также все любители техники и анализа данных. 

## Назначение

Первичный анализ структурированных данных с малым числом признаков. Например, автор данного проекта использует свои скрипты для домашних поделок с применением микросхем и микроконтроллеров (управляемый настольный штатив для фотоаппарата, пауэрбанк, цифровой микроскоп для пайки схем, простая подводная камера с корпусом из поликарбоната).

## Ограничения

Набор скриптов не подходит для неструктурированной информации (текст, изображения).

## Описание скриптов

В директории device расположена небольшая библиотека функций для расчётов (простые схемы), которая не использует внешних зависимостей. Для неё написан минимальный unit test. Остальные модули требуют внешние зависимости (только общеизвестные свободные модули). Пара функций для загрузки данных (API, база данных) в директории query. В директории plot скрипты для простого и быстрого визуального отображения, а простые вспомогательные скрипты для машинного обучения в директории ml.

## Установка AK-84 Tools

Установка не требуется, так как все мои программы поставляются в виде скриптов с открытым исходным кодом. Обратите внимание, что утилиты используют внешние библиотеки. Если у вас не создано виртуальное окружение Python, то это не беда. Выполните следующую команду в пустой директории, куда вы планируете добавить мои скрипты:

```bash
python3 -m venv venv
```

Далее нужно задействовать виртуальное окружение Python:

```bash
source venv/bin/activate
```

А после этого уже можно установить модули для анализа данных:

```bash
pip install numpy pandas matplotlib scikit-learn requests sqlalchemy psycopg2-binary
```

Установка внешних модулей завершена. Приятного использования.

Запрос информации у больших языковых моделей:

```python
import fetch_items


# Укажите URL
url = "http://localhost:11434/api/generate"

# Ваш массив объектов (входных данных)
data = [{
    "model": "gemma3:12b",
    "prompt": "Что такое вода? Ответ не больше 256 символов.",
    "stream": False
},
{
    "model": "qwen3:14b",
    "prompt": "Что такое вода? Ответ не больше 256 символов.",
    "stream": False
}]

# Последовательные запросы в один поток
results = fetch_items.fetch_items(url, data)

# Отображение результатов
for result in results:
    print(result["response"]["response"])
    print(result["response_time"])
```
